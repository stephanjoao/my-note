# Cadernos

Uma base de aplicativo tipo OneNote, só que mais simples:

- vários cadernos
- várias páginas por caderno
- canvas grande para organizar blocos
- blocos de texto
- blocos de tabela
- armazenamento local no navegador
- interface adaptada para celular e computador

## Como abrir

Basta abrir o arquivo `index.html` no navegador.

## Como usar

- clique em `Novo` para criar um caderno
- clique em `Nova` para criar uma página
- use `Texto` e `Tabela` para criar blocos
- arraste o topo de um bloco para mover
- arraste o fundo do canvas para navegar
- dê um clique duplo em um caderno ou página para renomear

## Observações

Os dados ficam salvos no `localStorage` do navegador. Em uma próxima etapa, a gente pode evoluir isso para:

- login
- sincronização em nuvem
- app instalável
- compartilhar cadernos
- exportação
